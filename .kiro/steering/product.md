# Product Overview

**Banka** — социальное приложение и каталог для коллекционеров энергетических напитков.

## Core Concept

Пользователи постят, оценивают и обсуждают банки от энергетиков, объединяются в группы/коллекции, сканируют штрих-коды для автозаполнения, ищут редкие лимитки.

## Key Features

- **Авторизация**: Google Sign-In через Firebase Auth
- **Профили**: Статистика коллекционера (количество банок, средняя редкость, топ-бренд)
- **Группы/Коллекции**: Публичные и приватные группы с участниками
- **Посты-банки**: До 6 фото, название, дата находки, редкость 1–9, теги
- **Социальные механики**: Лайки (транзакционные), комментарии (real-time)
- **Поиск**: По `searchKeywords` с фильтрами (rarity, brand, group)
- **Бренды**: Нормализованная коллекция брендов с витринами
- **Сканер штрих-кодов**: EAN-13/UPC lookup с автозаполнением и contribute-back

## Design Language

- **Тема**: Строгая тёмная (`#000000` фон)
- **Акценты**: Серые `#0F0F0F` / `#1A1A1A` / `#2A2A1A`, белый текст
- **Цвет акцента**: Янтарный `#FFB300`
- **Стиль**: Минималистичный, Material 3

## Target Audience

Коллекционеры энергетических напитков, которые хотят каталогизировать находки, делиться редкими банками, находить единомышленников.
